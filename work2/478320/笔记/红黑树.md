# 红黑树

红黑树不等于平衡二叉树，它可以有两层以上的差距，平衡二叉树虽然查找快，但是由于旋转占用的资源比较大，导致它添加数据的速度比较慢，红黑树有添加数据和查找都比较快的优点

红黑树遵循基本红黑规则

1. 每一个节点是红色或者黑色不能是其他颜色
2. 两个节点不能同时都是红色
3. 如果一个节点没有子节点或者父节点，则该节点的指针属性为Nil
4. 根节点必须是黑色
5. 对每一个节点，从该节点到其他所有后代节点的简单路径上，均包含相同数目的黑色节点
