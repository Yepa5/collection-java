## 项目功能：
* 对商品和订单表进行增删改查
* 购买商品，增加订单，同时商品库存减少
* 删除商品，同时相关订单取消
* 删除订单，同时恢复商品库存
* 按需对商品或订单进行排序

## 项目介绍：
使用mysql数据库
项目主要包括两个大类，对表的增删改查OperateTable以及对事务操作OperateTransaction
